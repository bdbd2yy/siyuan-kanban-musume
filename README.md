# siyuan-kanban-musume

<img src="./icon.png" width="160" height="160" alt="icon">

Let little lovely girl accompany you to write notes

## Core Features

- **Lightweight and concise**: Except for live2d, this project has no other dependencies
- **Change Model**: Supports changing models. Built-in models: `416`, `pio`, and `chino`.
- **More interaction**: Time greetings, welcome messages, idle tips, copy hints, touch hints, and visibility-return hints.
- **Toolbar actions**: include random message, model switching, screenshot, information, hide, and settings.
- **Settings panel**: set the position, dragging, idle tips, model selection, editable model list, and message JSON.
