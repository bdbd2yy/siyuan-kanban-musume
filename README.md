# SiYuan Kanban Musume

A local-first Live2D kanban companion for SiYuan.

## Features

- Local Cubism 2 models: `416`, `pio`, and `chino`.
- Time greetings, welcome messages, idle tips, copy hints, touch hints, and visibility-return hints.
- Toolbar actions for random message, model switching, screenshot, information, hide, and settings.
- Settings panel for position, dragging, idle tips, model selection, editable model list, and message JSON.

Settings are stored in `kanban-musume-setting.json`.
